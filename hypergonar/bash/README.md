## Environment variables

### Amazon S3

- SKY_AWS_PATH: Amazon S3 url path (domain/path)

- AWS_ACCESS_KEY_ID: Amazon S3 key id

- AWS_SECRET_ACCESS_KEY: Amazon S3 key secret

- AWS_DEFAULT_REGION: Amazon S3 region (eu-west-1)
